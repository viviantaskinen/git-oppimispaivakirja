# Oppimispäiväkirja: Paikallinen git

__Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet?__

Kirjoita tähän vastauksesi

Aloittaminen oli hieman vaikeaa. Sen jälkeen, kun aloin paremmin ymmärtämään mistä kyse, niin tekeminenkin nopeutui. Joitain kohtia tein uusiksi ja kokeilin varmistaakseni, että tein niin kuin pitää. Käytin tekoälyä paikoittain apuna, jos tuntui että olin jumissa. Lähtökohtaisesti pyrin kuitenkin selvittämään asiat omatoimisesti, jotta oppisin mahdollisimman paljon.

## Osiossa käyttämäni Git-komennot

| git init | uusi git repositorio |
| git add . | lisää talletukseen |
| git commit -m ""| tallentaa + kuvaus tapahtumasta |
|git status | näyttää tilan |
| git branch | näyttää kaikki haarat |
| git checkout + branch | haaranvaihto |
| git merge + branch | yhdistää haarat
| git push | lähettää githubiin nykyiset tallennetut tiedot |
