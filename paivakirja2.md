## Osiossa käyttämäni Git-komennot

| git clone + url | Kopioin repositorin |
| git push| lähetin githubiin tallennetut tiedot |
| git pull | fetch + merge |
| git fetch | lataa repositorista commitit ilman mergeä|
| git checkout + haara | vaihtaa toiseen haaraan |
| git checkout -b + haara | tämän onnistuin tekemään ja sitten selvittelin, missä meni vikaan! eli siis luo uuden haaraan ja avaa sen |
| git merge + branch | yhdistää haaran nykyisen haaran kanssa |
| git remote -v |tällä selvittelin, miksi en saanut tätä päiväkirjaa auki eli siis näyttää repositorien URL:t |
| git rm + tiedosto |poistin ylimääräisen paivakirja2 haaran, jonka takia en pystynyt avaamaan tätä|
